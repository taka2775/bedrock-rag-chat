# VPC の基本構成

## VPC とは

VPC（Virtual Private Cloud）は AWS 上に作成する仮想ネットワーク。EC2 や RDS などのリソースをこのネットワーク内に配置し、ネットワークレベルでのアクセス制御を行う。

## 主要コンポーネント

### サブネット

VPC 内の IP アドレスの範囲を分割したもの。用途に応じてパブリックサブネットとプライベートサブネットに分ける。

- **パブリックサブネット**: インターネットゲートウェイへのルートを持つサブネット。インターネットとの直接通信が可能。ALB や NAT Gateway を配置する
- **プライベートサブネット**: インターネットゲートウェイへの直接ルートを持たないサブネット。EC2, RDS, Lambda 等を配置する。インターネットへの通信は NAT Gateway 経由

### アベイラビリティゾーン（AZ）

高可用性のため、サブネットは複数の AZ に分散して配置する。一般的には 2 つ以上の AZ を使用する。

## ルートテーブル

サブネット内のトラフィックがどこに向かうかを定義するルールの集合。

- **パブリックサブネットのルートテーブル**: `0.0.0.0/0 → インターネットゲートウェイ`（インターネットへのトラフィックを IGW に転送）
- **プライベートサブネットのルートテーブル**: `0.0.0.0/0 → NAT Gateway`（インターネットへのトラフィックを NAT 経由で転送）
- **ローカルルート**: VPC 内の通信は自動的にルーティングされる（例: `10.0.0.0/16 → local`）

## インターネットゲートウェイ（IGW）

VPC とインターネットを接続するゲートウェイ。パブリックサブネットのリソースがインターネットと通信するために必要。VPC に 1 つだけアタッチできる。

## NAT Gateway

プライベートサブネットのリソースがインターネットにアクセスするためのゲートウェイ。インターネットからの着信接続は受け付けない（アウトバウンドのみ）。

- パブリックサブネットに配置する
- 高可用性のため、各 AZ に 1 つずつ配置するのがベストプラクティス
- コストがかかる（約 $0.045/時間 + データ転送料）。開発環境では 1 つの AZ にだけ配置してコスト削減も可能

## セキュリティグループ

リソース（ENI）単位のファイアウォール。インバウンド/アウトバウンドのトラフィックを許可ルールで制御する。

- **ステートフル**: インバウンドで許可したトラフィックの戻りは自動で許可される
- **デフォルト**: インバウンドは全拒否、アウトバウンドは全許可
- **許可ルールのみ**: 拒否ルールは設定できない（設定したいならネットワーク ACL を使う）

## ネットワーク ACL（NACL）

サブネット単位のファイアウォール。セキュリティグループとの違い:

- **ステートレス**: インバウンドとアウトバウンドを個別に設定する必要がある
- **許可・拒否の両方が可能**: 特定の IP アドレスをブロックするなどの用途に使える
- **ルール番号の順序で評価**: 番号が小さいルールから順に評価される

## 典型的な構成例

```
VPC (10.0.0.0/16)
├── パブリックサブネット AZ-a (10.0.1.0/24)
│   ├── ALB
│   └── NAT Gateway
├── パブリックサブネット AZ-c (10.0.2.0/24)
│   ├── ALB
│   └── NAT Gateway
├── プライベートサブネット AZ-a (10.0.11.0/24)
│   ├── EC2 / ECS
│   └── Lambda
├── プライベートサブネット AZ-c (10.0.12.0/24)
│   ├── EC2 / ECS
│   └── Lambda
├── プライベートサブネット（DB）AZ-a (10.0.21.0/24)
│   └── RDS
└── プライベートサブネット（DB）AZ-c (10.0.22.0/24)
    └── RDS
```

## VPC エンドポイント

VPC 内からAWS サービス（S3, DynamoDB 等）にインターネットを経由せずにアクセスするための機能。

- **ゲートウェイエンドポイント**: S3 と DynamoDB のみ対応。無料
- **インターフェースエンドポイント（PrivateLink）**: ほぼすべての AWS サービスに対応。ENI を作成するため時間あたりのコストが発生
